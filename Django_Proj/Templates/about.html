About PleasantSounds

The idea originated from the fact that not everyone can study music theory to create music. Many have the desire to create and innovate but with limited knowledge there is not much they can do. We believe creativity knows no boundry. To nuture the musical ideas and mould it into executable objects, this is the reason why we started this project.

PleasantSounds uses AI model to predict the genre of music. The file can be uploaded in the classify section. We bring the intersection of music and technology. The model was trained on GTZAN dataset from kaggle. The latest model uses LSTM which is considered one of the best options for processing audio files.

Background

We started this project with little knowledge of web development and ML models.
